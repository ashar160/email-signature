## Installation Instructions by Email

### **Gmail (Web)**

1. Open Gmail and click the **Settings gear icon** (⚙️) → **See all settings**
2. Go to **Forwarding and POP/IMAP** tab
3. Scroll down to **Signature** section
4. Click **Create new**
5. Name your signature: `Villas Pinares - Professional`
6. Open the HTML file in a text editor
7. Copy ALL the code from `<table cellpadding...` to `</table>` (just the signature table, NOT the full HTML)
8. Paste it into the signature box
9. Click **Save changes**

**Note:** Gmail converts HTML to its own format, so the visual appearance may vary slightly.

---

### **Outlook (Desktop - Windows)**

1. Open Outlook and go to **File** → **Options**
2. Click **Mail** → **Signatures**
3. Click **New** and name it: `Villas Pinares - Professional`
4. Click in the signature editing area
5. Go to **Insert** → **Signature** (or use Insert menu to add HTML)
6. For best results, use **Insert > Text from File** and select the HTML file
7. Click **OK** → **Save**

**Alternative Method (Insert as HTML):**
- Copy the HTML signature code
- Paste it directly into the signature field
- Click **OK** → **Save**

---

### **Outlook (Web - Office 365)**

1. Click the **Settings gear icon** (⚙️) at the top right
2. Select **View all Outlook settings**
3. Go to **Mail** → **Compose and reply**
4. Scroll to **Email signature**
5. Open the HTML file in a text editor
6. Copy the HTML code
7. Click in the signature box and paste the HTML
8. Click **Save**

**Note:** Web-based Outlook has limited HTML support; test after saving.

---

### **Apple Mail (Mac)**

1. Open **Mail** → **Preferences** (⌘,)
2. Click the **Signatures** tab
3. Click **+** to create a new signature
4. Name it: `Villas Pinares - Professional`
5. Close preferences
6. In Mail, click **Mail** → **Preferences** again
7. Go to **Compose** tab and select your new signature as default
8. Paste the HTML content directly (Apple Mail has limited HTML support)

---

### **Apple Mail (iOS/iPadOS)**

1. Go to **Settings** → **Mail** → **Signature**
2. Add your signature text (iOS doesn't support HTML signatures)
3. Manually recreate the signature with plain text formatting or use line breaks
